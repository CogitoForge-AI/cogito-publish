# Hướng dẫn theo việc

Các công thức định hướng theo công việc. Mỗi hướng dẫn giả định bạn đã có một
trang chạy được (xem [tutorial](../tutorial/build-your-first-site.md)) và giải
quyết một bài toán cụ thể.

- [Thêm quốc tế hóa (i18n)](internationalization.md) - phục vụ cùng một trang ở
  nhiều ngôn ngữ.
- [Tùy biến theme](customize-theme.md) - tách (eject) một theme tích hợp và sửa
  template cùng CSS của nó.
- [Sinh API reference](api-reference.md) - xuất bản mục `References` từ docstring
  Python với plugin `apidoc`.
- [Viết plugin riêng](write-a-plugin.md) - gắn vào một hook để thêm hành vi của
  riêng bạn.
- [Triển khai trang đã build](deploy.md) - xuất bản thư mục `dist/`.
