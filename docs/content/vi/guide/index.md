# Hướng dẫn

Tài liệu thực hành, bắt tay vào việc để dựng và chạy một trang với pyssg.

- [Bắt đầu nhanh](/vi/guide/getting-started/) - cài đặt, tạo khung và build trang
  đầu tiên của bạn.
- [Cấu hình](/vi/guide/configuration/) - file `pyssg.config.py` và đối tượng
  `Config`.
- [Dev server](/vi/guide/dev-server/) - `pyssg serve` để xem trước trực tiếp với
  rebuild-khi-thay-đổi.
- [Triển khai](/vi/guide/deploy/) - công thức cho GitHub Pages, Netlify và
  Cloudflare Pages.

Nếu bạn muốn nắm bức tranh khái niệm trước, hãy nhảy tới mục
[Kiến trúc](/vi/architecture/).
