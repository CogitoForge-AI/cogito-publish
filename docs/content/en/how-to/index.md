# How-to guides

Task-oriented recipes. Each guide assumes you already have a working site (see the
[tutorial](../tutorial/build-your-first-site.md)) and solves one specific problem.

- [Add internationalization (i18n)](internationalization.md) - serve the same
  site in more than one language.
- [Customize a theme](customize-theme.md) - eject a built-in theme and edit its
  templates and CSS.
- [Generate an API reference](api-reference.md) - publish a `References` section
  from your Python docstrings with the `apidoc` plugin.
- [Write a custom plugin](write-a-plugin.md) - tap a hook to add your own
  behaviour.
- [Deploy a built site](deploy.md) - publish the `dist/` directory.
